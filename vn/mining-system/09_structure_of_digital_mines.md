# 09 Cấu trúc mỏ kỹ thuật số

Trong CryptoStone, mỗi stone được xem như một mỏ kỹ thuật số độc lập.

| Pool | Mô tả |
| ---- | ----- |
| Garnet Pool | Khai thác Garnet NFT |
| Amethyst Pool | Khai thác Amethyst NFT |
| Aquamarine Pool | Khai thác Aquamarine NFT |
| Diamond Pool | Khai thác Diamond NFT |
| Emerald Pool | Khai thác Emerald NFT |
| Pearl Pool | Khai thác Pearl NFT |
| Ruby Pool | Khai thác Ruby NFT |
| Spinel Pool | Khai thác Spinel NFT |
| Sapphire Pool | Khai thác Sapphire NFT |
| Opal Pool | Khai thác Opal NFT |
| Topaz Pool | Khai thác Topaz NFT |
| Zircon Pool | Khai thác Zircon NFT |

Mỗi mining pool sử dụng cùng một STONX Token, nhưng có điều kiện khai thác khác nhau.

| Hạng mục | Ý nghĩa |
| -------- | ------- |
| Stone Type | Loại đá quý được khai thác từ pool |
| Max Supply | Số lượng phát hành tối đa của stone đó |
| Base Mining Interval | Chu kỳ khai thác cơ bản |
| Target Pool Power | Mining Power tham chiếu |
| Current Pool Power | Mining Power hiện đang tích lũy trong pool |
| Minted Supply | Số lượng đã được khai thác |
| Pool Difficulty | Độ khó dựa trên tổng mức tham gia |
| Scarcity Multiplier | Hệ số khan hiếm dựa trên halving |

Giống như mỏ kim cương và mỏ ruby ngoài đời thực có trữ lượng khác nhau, mỗi stone trong CryptoStone có tổng số lượng và độ khó khai thác riêng. Việc khai thác nhiều Diamond không làm tăng độ khó của Ruby. Việc Ruby gần cạn nguồn cung cũng không làm giảm nguồn cung của Sapphire.

Mỗi stone có trữ lượng kỹ thuật số và cấu trúc khai thác độc lập của riêng mình.
