# 32 Rarity tier structure

CryptoStone rarity tiers को पाँच grades में统一 कर सकता है:

| Tier | Meaning |
| --- | --- |
| Common | सबसे सामान्य result |
| Rare | सामान्य से बेहतर combination |
| Epic | स्पष्ट रूप से दुर्लभ |
| Legendary | अत्यंत दुर्लभ |
| Genesis | सर्वोच्च collectible tier |

इन tiers को attribute score ranges और probability balance से निर्धारित किया जाना चाहिए।

Design goal:

* Common tier पर्याप्त मात्रा में हो ताकि ecosystem activity बने
* Rare और Epic users को meaningful upside दें
* Legendary और Genesis बहुत कम probability वाले रहें
* उच्च tier results market में naturally scarce रहें

यदि कोई उपयोगकर्ता 36 के आसपास claims करता है, तो probability expectation के आधार पर कम से कम एक meaningful higher-tier result मिलने की संभावना बन सकती है, लेकिन guarantee नहीं होनी चाहिए।
