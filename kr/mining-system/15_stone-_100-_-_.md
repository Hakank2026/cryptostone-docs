# 15 STONX 배분 구조

CryptoStone에서 STONX는 디지털 보석 채굴에 참여하기 위한 단일 마이닝 자원이다. 개발자는 STONX의 배포 구조가 특정 내부자에게 사전적으로 권리를 부여하는 방식보다, 누구나 동일한 조건에서 접근할 수 있는 공개 시장 구조를 따르는 것이 CryptoStone의 철학에 부합한다고 본다.

따라서 STONX는 팀 물량, 창업자 리저브, 비공개 내부자 배정 없이 전체 공급량을 시장 공개형 배분 구조로 설계한다. 전체 공급량은 DEX 유동성, 다이렉트 세일, 전략 생태계 파트너, 거래소 유동성 준비에 배정된다.

| 항목 | 구조 |
| ---- | ---- |
| Total Supply | 1,200,000,000 STONX |
| Team Allocation | 없음 |
| Founder Reserve | 없음 |
| Hidden Insider Allocation | 없음 |
| Additional Mint | 없음 |
| Primary Utility | Gem NFT 채굴을 위한 마이닝 풀 참여 |
| Mechanism | STONX 취득 → 스테이킹 → Mining Power 생성 → PoM 누적 → Gem NFT claim |
| Supply Trust | 발행량, 풀 구조, 확률표, claim 조건은 컨트랙트와 공개 데이터로 검증 가능 |

## 15.1 Allocation

| Allocation | Ratio | TGE Unlock | Lock / Vesting | 명분 |
| ---------- | ----: | ---------- | -------------- | ---- |
| DEX Liquidity Reserve | 50% | 초기 유동성에 필요한 범위 내 단계 투입 | LP 장기 락업 또는 소각, 잔여분은 유동성 확장 시 단계 사용 | 누구나 접근 가능한 온체인 진입 경로를 가장 크게 확보 |
| Direct Sale Pool | 20% | 판매 컨트랙트 한도 내 공개 판매 | 미판매 물량은 풀에 보관하고 단계적으로 공급 | 홈페이지에서 채굴 참여용 STONX를 직접 확보할 수 있는 경로 제공 |
| Strategic / VC Allocation | 10% | 0% 또는 제한적 초기 언락 | 6개월 cliff + 24개월 선형 vesting | 거래소, 인프라, 파트너십 확장을 위한 전략 물량 |
| Listing / Liquidity Reserve | 20% | 상장 준비와 마켓메이킹 필요 범위 내 사용 | 공개 지갑 관리, 목적 외 사용 제한, 단계적 집행 | 중앙화 거래소 유동성, 캠페인, 시장 안정성 준비 |

DEX Liquidity Reserve를 가장 크게 배정하는 이유는 STONX의 핵심 접근 경로가 공개 시장이어야 하기 때문이다. Direct Sale Pool은 신규 사용자가 복잡한 거래 경로 없이 마이닝 참여에 필요한 STONX를 확보하도록 돕는다. Strategic / VC Allocation은 생태계 확장 파트너에게 제한적으로 사용되며, 장기 vesting을 통해 단기 매도 압력을 줄인다. Listing / Liquidity Reserve는 상장, 마켓메이킹, 캠페인, 시장 안정성 준비에만 사용되는 목적성 리저브다.

사용자는 공개 시장 또는 다이렉트 세일 경로에서 STONX를 취득한 뒤, 원하는 스톤 마이닝 풀에 이를 투입할 수 있다. 이후 사용자는 스테이킹한 STONX를 기반으로 Mining Power를 생성하고, 시간이 지남에 따라 Proof of Mining, PoM 값을 누적한다. 누적된 PoM 값이 해당 풀의 필요 임계값에 도달하면 Gem NFT를 claim할 수 있다.

이 구조에서 DEX와 Direct Sale은 STONX에 접근하기 위한 공개 진입 경로이며, 마이닝 풀은 STONX의 실제 사용처이다. 즉, STONX의 목적은 단순 보유나 투기적 유통이 아니라, 디지털 보석 채굴에 참여하기 위한 프로토콜 자원으로 기능하는 데 있다.

공개형 배분 구조가 신뢰를 얻기 위해서는 초기 유동성 공급 방식, LP 처리 방식, 컨트랙트 권한, 추가 발행 불가 여부, 주요 리저브 지갑이 명확히 공개되어야 한다. 초기 LP 토큰의 처리 방식은 장기 락업 또는 소각 방식으로 공개되는 것이 바람직하다. 이는 유동성 회수에 대한 우려를 줄이고, STONX의 공개 시장 접근 구조에 대한 신뢰를 높이기 위한 것이다.

CryptoStone의 신뢰는 특정 주체의 특권적 배정 물량이 아니라, 고정된 총 발행량, 팀 물량 없음, 변경 불가능한 채굴 규칙, 검증 가능한 PoM 구조, 그리고 온체인 데이터의 투명성에 의해 형성된다.
