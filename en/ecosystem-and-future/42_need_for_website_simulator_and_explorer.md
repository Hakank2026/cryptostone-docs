# 42\_need\_for\_website\_simulator\_and\_explorer

The CryptoStone protocol itself operates through smart contracts and on-chain data. However, a separate user interface is needed for users to intuitively understand and participate in it.

Therefore, the CryptoStone ecosystem may require the following web-based tools:

| Tool                 | Role                                                                                                                                                  |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| Mining Simulator     | Calculates estimated mining time based on the user’s STONX amount, selected pool, and lock-up period.                                                 |
| PoM Dashboard        | Displays the user’s PoM value by pool, required PoM threshold, and claim availability.                                                                |
| Pool Dashboard       | Displays mined quantity, remaining supply, Pool Difficulty, and Scarcity Multiplier for each STONX pool.                                              |
| Gem Explorer         | Allows users to view attributes, rarity, tokenId, mining time, and transaction history of mined Gem NFTs.                                             |
| Rarity Explorer      | Allows users to check rarity and Probability Rarity Index by Weight, Color, Clarity, and Cut combinations.                                            |
| Refinery Console     | Allows users to check and execute Gem Refinement conditions, probability tables, and refinement records.                                              |
| Protocol Status Page | Shows key protocol information such as Open Market Allocation, No Admin Mint, total supply, probability tables, and pool status in a verifiable way. |

These tools do not replace the protocol’s core trust foundation. They serve as interfaces that help users easily understand on-chain data. In other words, the essence of CryptoStone lies in contracts and on-chain rules, while the website and explorer are tools for interpreting and visualizing them.
