# 32\_稀有度等级结构

完美的顶级组合只能以极低概率生成。从稀缺性角度来看，这是优点，但在收藏市场中，需要形成多样化的高等级层级，交易与收藏活动才会更加活跃。

因此，CryptoStone 可以基于 Rarity Score 与 Probability Rarity Index 适用以下稀有度等级：

| Tier      | 示例标准         | 含义          |
| --------- | ------------ | ----------- |
| Common    | 以普通属性为主      | 最常见的宝石群     |
| Rare      | 至少 1 个高级属性   | 一般稀有宝石      |
| Epic      | 至少 2 个高级属性   | 具有较高收藏价值的宝石 |
| Legendary | 至少 3 个高级属性   | 非常稀有的宝石     |
| Genesis   | 极端组合或早期挖矿唯一性 | 顶级收藏资产      |

Genesis Tier 不一定仅由分数决定。早期挖矿时间、较低 tokenId、极端概率组合、特定宝石的稀缺性等因素都可能被综合考虑。

该结构类似于宝可梦卡牌、体育卡牌、限量收藏品市场中，稀有等级形成收藏价值的方式。不过在 CryptoStone 中，稀有度并不是由运营方主观判断，而是由公开概率表和链上属性值计算得出。
