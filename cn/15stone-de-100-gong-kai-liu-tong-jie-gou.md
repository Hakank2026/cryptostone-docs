# 15\_STONE\_的\_100\_公开流通结构

在 CryptoStone 中，STONE 是参与数字宝石挖矿的单一挖矿资源。本文作者认为，与其采用向特定内部人预先赋予权利的分配方式，STONE 的分配结构更应遵循任何人都可以在相同条件下访问的公开市场结构，这更符合 CryptoStone 的哲学。

因此，STONE 遵循 100% 公开流通结构。这意味着 STONE 并不是预先分配给特定主体，而是通过公开 DEX 流动性进入市场流通，任何人都可以在相同公开市场条件下取得。

| 项目                     | 结构                                                       |
| ---------------------- | -------------------------------------------------------- |
| Total Supply           | 1,200,000,000 STONE                                      |
| Distribution Principle | 100% 公开流通                                                |
| Market Access          | 公开 DEX 流动性                                               |
| Additional Mint        | 无                                                        |
| Access Rule            | 任何人都可在相同公开市场条件下取得                                        |
| Primary Utility        | 参与 Gem NFT 挖矿池                                           |
| Mechanism              | 获取 STONE → 质押 → 生成 Mining Power → 累积 PoM → claim Gem NFT |
| Supply Trust           | 发行量、矿池结构、概率表、claim 条件均可由合约验证                             |

用户可以在公开市场获得 STONE 后，将其投入自己选择的宝石挖矿池。之后，用户基于质押的 STONE 生成 Mining Power，并随着时间流逝累积 Proof of Mining，即 PoM。当累积的 PoM 达到对应矿池的所需阈值后，用户即可 claim Gem NFT。

在这一结构中，DEX 是进入 STONE 的公开入口，而挖矿池则是 STONE 的实际使用场景。也就是说，STONE 的目的并不是单纯持有或投机流通，而是在数字宝石挖矿中作为协议资源发挥作用。

基金会或早期生态贡献者也应通过公开市场在相同条件下取得 STONE，而非通过单独的预先分配。该方式旨在最大程度减少特定主体的先占结构，并使协议参与标准与公开市场和链上规则保持一致。

为了让 100% 公开流通结构获得信任，初始流动性供应方式、LP 代币处理方式、合约权限以及是否无法追加发行都必须被明确公开。初始流动性通过公开 DEX 池形成，初始 LP 代币的处理方式最好以长期锁定或销毁方式公开。这有助于降低对流动性撤回的担忧，并增强对 STONE 公开市场访问结构的信任。

即使基金会或早期贡献者取得 STONE，也应在公开市场以相同条件取得，而不应通过单独的预分配方式取得。主要持有钱包也最好以可链上确认的方式公开。该结构并不是否定基金会的存在，而是表达基金会也应在公开市场和相同规则之内参与，而不是通过协议外的特权分配参与。

CryptoStone 的信任并不来自某个特定主体的预分配份额，而是来自固定总发行量、100% 公开流通结构、不可变更的挖矿规则、可验证的 PoM 结构以及链上数据透明性。
