# 15\_STONEの100パーセント公開流通構造

CryptoStoneにおいて、STONEはデジタル宝石採掘に参加するための単一マイニング資源である。筆者は、STONEの配布構造が特定内部者に事前的権利を付与する方式よりも、誰でも同一条件でアクセスできる公開市場構造に従うことがCryptoStoneの哲学に合致すると考える。

したがって、STONEは100%公開流通構造に従う。これは、STONEが特定主体に事前配分される方式ではなく、公開DEX流動性を通じて市場に流通し、誰でも同一の公開市場条件で取得できることを意味する。

| 項目                     | 構造                                                        |
| ---------------------- | --------------------------------------------------------- |
| Total Supply           | 1,200,000,000 STONE                                       |
| Distribution Principle | 100%公開流通                                                  |
| Market Access          | 公開DEX流動性                                                  |
| Additional Mint        | なし                                                        |
| Access Rule            | 誰でも同一の公開市場条件で取得可能                                         |
| Primary Utility        | Gem NFT採掘のためのマイニングプール参加                                   |
| Mechanism              | STONE取得 → ステーキング → Mining Power生成 → PoM蓄積 → Gem NFT claim |
| Supply Trust           | 発行量、プール構造、確率表、claim条件はコントラクトで検証可能                         |

ユーザーは公開市場でSTONEを取得した後、希望するストーンマイニングプールにこれを投入できる。その後、ユーザーはステーキングしたSTONEに基づきMining Powerを生成し、時間の経過とともにProof of Mining、PoM値を蓄積する。蓄積されたPoM値が当該プールの必要閾値に到達すると、Gem NFTをclaimできる。

この構造において、DEXはSTONEへアクセスするための公開入口であり、マイニングプールはSTONEの実際の使用先である。つまり、STONEの目的は単純保有や投機的流通ではなく、デジタル宝石採掘に参加するためのプロトコル資源として機能することにある。

財団または初期エコシステム貢献者も、別途の事前配分ではなく、公開市場で同一条件によりSTONEを取得できる。この方式は、特定主体による先占構造を最小化し、プロトコル参加の基準を公開市場とオンチェーン規則に合わせるためのものである。

100%公開流通構造が信頼を得るためには、初期流動性供給方式、LP処理方式、コントラクト権限、追加発行不可の有無が明確に公開されなければならない。初期流動性は公開DEXプールを通じて形成され、初期LPトークンの処理方式は長期ロックアップまたはバーン方式として公開されることが望ましい。これは流動性回収への懸念を減らし、STONEの公開市場アクセス構造に対する信頼を高めるためである。

財団または初期貢献者がSTONEを取得する場合にも、別途の事前配分ではなく公開市場で同一条件により取得すべきであり、主要保有ウォレットはオンチェーンで確認可能な形で公開することが望ましい。この構造は財団の存在を否定するためのものではなく、財団もまたプロトコル外部の特権的配分ではなく、公開市場と同一の規則の中で参加すべきであるという原則を表現する。

CryptoStoneの信頼は、特定主体の事前配分量ではなく、固定された総発行量、100%公開流通構造、変更不可能な採掘規則、検証可能なPoM構造、そしてオンチェーンデータの透明性によって形成される。
