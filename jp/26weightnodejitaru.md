# 26\_Weight属性のデジタル化

重量は、すべてのストーンで共通して CT、すなわちカラット単位を使用します。ただし、各宝石の重量値はランダムに生成されます。

* Minimum Weight = 0.10 CT
* Maximum Weight = 200.00 CT
* Storage Unit = 0.01 CT

スマートコントラクトでは小数点を直接保存せず、0.01 CT 単位の整数として保存します。

| 表示重量      | コントラクト保存値 |
| --------- | --------: |
| 0.10 CT   |        10 |
| 1.25 CT   |       125 |
| 10.50 CT  |      1050 |
| 200.00 CT |     20000 |

重量は均等確率で生成されません。現実の宝石において大きなカラットほど希少であるように、CryptoStone でも大きな重量ほど低い確率で生成されます。

| 1.00 \~ 1.99 CT     |      25.00% | Uncommon    |     5 |

| 5.00 \~ 9.99 CT     |       6.00% | Epic        |    13 |
| 10.00 \~ 19.99 CT   |       2.50% | Legendary   |    17 |

| 50.00 \~ 99.99 CT   |       0.40% | Ancient     |    23 |
