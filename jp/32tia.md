# 32\_希少度ティア構造

完全な最上位組み合わせは、極めて低い確率でのみ生成され得る。これは希少性の観点では利点であるが、収集市場では多様な上位ティアが形成されてこそ、取引と収集活動がより活発になり得る。\
したがって、CryptoStoneはRarity Score と Probability Rarity Index を基に、次のような希少度ティアを適用できる。

| Tier      | 基準例                | 意味        |
| --------- | ------------------ | --------- |
| Common    | 一般属性中心             | 最も一般的な宝石群 |
| Rare      | 上位属性1つ以上           | 一般的な希少宝石  |
| Epic      | 上位属性2つ以上           | 収集価値の高い宝石 |
| Legendary | 上位属性3つ以上           | 非常に希少な宝石  |
| Genesis   | 極端な組み合わせまたは初期採掘固有性 | 最上位収集資産   |

Genesis Tier は、単純にスコアだけで決定されない場合がある。初期採掘時点、低い tokenId、極端な確率組み合わせ、特定ストーンの希少性などが複合的に考慮され得る。

この構造は、ポケモンカード、スポーツカード、限定版コレクティブル市場において希少等級が収集価値を形成する方式と類似している。ただし、CryptoStone では希少度が運営者の主観的判断ではなく、公開された確率表とオンチェーン属性値によって計算される点に違いがある。
